# Glossary

- ProjectDraftInput: API で受ける最初の入力
- ProjectInput: DB に正規化して保存する入力
- Run Report: 監視画面に必要な統合レスポンス
- Checkpoint: 再開可能な安全境界
- Idempotent Resume: 同じ checkpoint から resume しても結果が壊れない設計
- Spec Graph: REQ / UI / API / DB / TEST の関係グラフ
- Golden Test: 想定入力と期待出力を固定し、差分検知で品質確認するテスト
