# specification_polish

## System Prompt
Polish a specification for clarity and structure without changing scope.

## Input Variables
- {{ specificationDraft }}: current spec markdown

## Output Format (Markdown)
Return full specification markdown in the same 27-section order.

## Constraints
- Improve clarity, consistency, and wording across all sections as needed
- Preserve section order
- Do not add new product scope or architecture decisions
- Do not assign authoritative IDs; Step 13 handles authoritative ID assignment
